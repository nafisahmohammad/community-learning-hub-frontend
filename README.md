
# Community Learning Hub Frontend

A React (Vite) frontend for the Community Learning Hub platform. Users can browse hubs and courses, enroll, and track learning progress.

## Tech Stack
- React + Vite
- React Router
- Tailwind CSS
- Axios

## Pages (Planned)
- Landing (/)
- Courses (/courses)
- Dashboard (/dashboard)

## Getting Started
```bash
npm install
npm run dev
